# OpenCharade
Open charade is an open source collabaritve charade game app for iOS, Android and Windows.
People can create their own decks quick and easy and share them with the whole world - not like other charade type apps where those need to be purchased for money.

## Roadmap
1. (Work in progress)Player can create their own decks and play 
2. (Future) Player can upload their decks and download other decks
